## Troubleshooting

**Blank screen?**
- Check server running: `cd server && npm start`
- Verify port 5174: `curl localhost:5174`

**Styles wrong?**
- Rebuild: `cd client && npm run build`
- Clear cache: Ctrl+Shift+R

**Can't enable?**
- Check file: `grep AppPresentation client/src/main.tsx`
- Should see: `import { App } from './AppPresentation.js';`
